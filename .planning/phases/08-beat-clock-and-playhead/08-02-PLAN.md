Extend drawLaneCanvas with an optional playheadBeat parameter that draws a solid 2px #6366f1 vertical line at the correct beat-modulo position. Add a RAF loop to AnimationPanel that runs only while isPlaying is true — each frame it reads the current beat, computes each lane's playhead x, and redraws the entire lane canvas (curve + playhead together). **Step 4 — Add ref map and RAF loop to AnimationPanel.**

Inside the `AnimationPanel` function, add the following after the existing state/ref declarations (after `dragStartHeight` ref, before the drag-handle handlers):

```typescript
// Ref map: property → canvas element — populated by AnimLane registration callbacks (D-02)
const laneCanvasRefs = useRef<Map<AnimatableProperty, HTMLCanvasElement>>(new Map())

// Stable callback so AnimLane useEffect deps don't fire on every render
const handleCanvasRef = useCallback(
  (property: AnimatableProperty, el: HTMLCanvasElement | null) => {
    if (el) {
      laneCanvasRefs.current.set(property, el)
    } else {
      laneCanvasRefs.current.delete(property)
    }
  },
  []
)

// Read isPlaying via React hook so useEffect re-runs on transition (D-04)
const isPlaying = usePlaybackStore((s) => s.isPlaying)

// RAF loop — redraws all lane canvases every frame while playing (D-02, D-03, D-04)
useEffect(() => {
  let rafId: number

  function tick() {
    const { bpm } = playbackStore.getState()
    const beat = getCurrentBeat(bpm)

    for (const [prop, canvas] of laneCanvasRefs.current.entries()) {
      const ctx = canvas.getContext('2d')
      if (!ctx) continue
      const curve = animationStore.getState().curves[shape?.id ?? '']?.[prop]
      if (!curve) continue
      drawLaneCanvas(ctx, canvas.width, canvas.height, curve, prop, null, beat)
    }

    rafId = requestAnimationFrame(tick)
  }

  if (isPlaying) {
    // Start RAF loop
    rafId = requestAnimationFrame(tick)
  } else {
    // Stopped: draw each lane once at beat 0 (D-04, D-05)
    for (const [prop, canvas] of laneCanvasRefs.current.entries()) {
      const ctx = canvas.getContext('2d')
      if (!ctx) continue
      const curve = animationStore.getState().curves[shape?.id ?? '']?.[prop]
      if (!curve) continue
      drawLaneCanvas(ctx, canvas.width, canvas.height, curve, prop, null, 0)
    }
  }

  return () => {
    if (rafId !== undefined) cancelAnimationFrame(rafId)
  }
}, [isPlaying, shape])
```

Important implementation rules:
- `playbackStore.getState()` is called INSIDE the RAF callback (vanilla Zustand read, NOT a hook) — this is correct and does not trigger re-renders (UI-SPEC React State Interaction Contract).
- `getCurrentBeat(bpm)` is called each frame — pure, no state.
- `selectedPointIdx` is passed as `null` in the RAF redraw path. The per-lane `useEffect` in AnimLane still runs on React renders and handles selected-point highlighting with the correct selectedPointIdx. The RAF loop does not need to know the selected index because it runs concurrently with React renders (the next React render will re-draw with the correct selection state on any interaction).
- The `isPlaying` transition is observed via the React hook `usePlaybackStore` so the `useEffect` re-runs exactly when playback starts or stops.
- Do NOT call `setState` or trigger any React state update from inside `tick()`.
