**Ghost pass block** — insert AFTER each primary drawLaneCanvas call, BEFORE the next `rafId = requestAnimationFrame(tick)` (in RAF) or BEFORE the loop's closing brace (in stopped-state):

```typescript
// Ghost repetitions — fill viewport when zoomBeats > curve.duration (ANIM-09, D-08, D-09)
// Read canvas dimensions fresh each iteration (Pitfall 2: avoid stale cached h)
const cw = canvas.width
const ch = canvas.height
const primaryWidthPx = (curve.duration / zoomBeats) * cw
const repeatCount = Math.floor(zoomBeats / curve.duration) - 1

// Full ghost copies (i = 1 .. repeatCount)
for (let i = 1; i <= repeatCount; i++) {
  const ghostStartPx = (i * curve.duration / zoomBeats) * cw
  ctx.save()
  ctx.globalAlpha = 0.30
  ctx.beginPath()
  ctx.rect(ghostStartPx, 0, primaryWidthPx, ch)
  ctx.clip()
  ctx.translate(ghostStartPx, 0)
  // Ghost callers pass undefined for zoomBeats — drawLaneCanvas uses curve.duration as denominator
  // to fill the ghost's local canvas width proportionally (D-09)
  drawLaneCanvas(ctx, primaryWidthPx, ch, curve, prop, null, undefined, undefined)
  ctx.restore()
}

// Partial ghost at right edge when zoomBeats % curve.duration !== 0 (Claude's Discretion)
const remainder = zoomBeats % curve.duration
if (remainder > 0 && repeatCount >= 0) {
  const partialStartPx = Math.floor(zoomBeats / curve.duration) * (curve.duration / zoomBeats) * cw
  const partialWidthPx = (remainder / zoomBeats) * cw
  ctx.save()
  ctx.globalAlpha = 0.30
  ctx.beginPath()
  ctx.rect(partialStartPx, 0, partialWidthPx, ch)
  ctx.clip()
  ctx.translate(partialStartPx, 0)
  drawLaneCanvas(ctx, primaryWidthPx, ch, curve, prop, null, undefined, undefined)
  ctx.restore()
}
```

**CRITICAL implementation notes:**

1. In the RAF tick, `zoomBeats` is the value from the React hook (available in the closure because it is in the `useEffect` deps array after Plan 01). Use it directly.

2. In the stopped-state draw (the `else` branch), `zoomBeats` is also available from the hook closure. Use the same variable. No need to call `uiStore.getState()` separately — it is already in scope.

3. The ghost pass must be inside the `for (const [prop, canvas] of laneCanvasRefs.current.entries())` loop, after the primary `drawLaneCanvas` call for that lane. Each lane gets its own ghost passes independently.

4. Do NOT call `ctx.clearRect` anywhere in the ghost pass sequence. The primary `drawLaneCanvas` call already cleared and drew the background. The ghost passes draw on top (this is correct per Pitfall 4).

5. Read `canvas.width` and `canvas.height` at the start of each loop iteration (using `const cw = canvas.width; const ch = canvas.height`), not once before the loop (Pitfall 2).

6. Each ghost pass MUST be wrapped in `ctx.save()` / `ctx.restore()` (Pitfall 3: translations are cumulative without restore).
