## Overview

Full end-to-end Shape Music Sequencer PoC. Users place geometric shapes on a 4×4 grid canvas; each shape is an autonomous Web Audio voice whose color, type, size, and spline animation curves map directly to oscillator pitch, timbre (WaveShaper), filter, amplitude, and stereo position. Playback controls, key/scale selector, and a multi-lane spline animation panel complete the v1 instrument.

## Milestone Summary

**Decisions:**

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| All shapes play continuously (no playhead trigger) | Animations drive sound evolution — core mechanic | ✓ Good |
| Strict grid (4×4) | Predictable timing, manageable polyphony for PoC | ✓ Good |
| Click → side panel for properties | Full per-shape control without cluttering canvas | ✓ Good |
| Saturation → WaveShaper harmonic richness (not reverb) | HSV metaphor; no mud at 32 voices; makeDistortionCurve existed | ✓ Good |
| LFO → spline curves (full replacement, D-05) | Two subsystems = maintenance trap; BeatFraction maps to loop duration | ✓ Good |
| frozenBeatPos freeze-on-stop pattern | Module variable in canvasEngine; O(1) overhead; single write path | ✓ Good |
| Web Audio API (no Tone.js) | Direct control; no third-party abstraction needed for PoC complexity | ✓ Good |
