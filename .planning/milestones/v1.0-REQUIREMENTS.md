# Requirements Archive: v1.0 MVP

## v1 Requirements — Final Status

### Canvas

- [x] **CANV-01**: User can place a shape on any empty grid cell (4x4 grid, up to 16 cells) — *Validated Phase 1*
- [x] **CANV-02**: User can click a placed shape to open the property edit panel — *Validated Phase 3*
- [x] **CANV-03**: User can remove a placed shape from the canvas — *Validated Phase 3*

### Shapes

- [x] **SHPE-01**: Circle shape renders on the canvas and generates a sine wave oscillator voice — *Validated Phase 2*
- [ ] **SHPE-06**: Multiple shapes per cell supported for complex timbres — *Deferred (not implemented in v1)*

### Color Mapping

- [x] **COLR-01**: Shape hue maps to pitch (oscillator frequency) — *Validated Phase 2 (direct linear hue→frequency mapping, quick-260424-2px)*
- [x] **COLR-02**: Shape saturation maps to harmonic richness (WaveShaperNode two-stage Chebyshev + soft-clip) — *Validated Phase 6*
- [x] **COLR-03**: Shape value/brightness maps to filter cutoff frequency — *Validated Phase 2*

### Audio Engine

- [x] **AUDI-01**: All placed shapes play continuously and simultaneously when playback is active — *Validated Phase 2*
- [x] **AUDI-02**: Each shape is an independent audio voice with its own oscillator, WaveShaper, and filter chain — *Validated Phase 2*
- [x] **AUDI-03**: Cell column position maps to stereo pan (StereoPannerNode) — *Validated Phase 6*

### Animation

- [ ] **ANIM-01**: Shape size oscillates at a configurable rate (amplitude LFO modulation) — *Superseded: LFO replaced by spline curves in Phase 7 (D-04/D-05)*
- [x] **ANIM-02**: Per-property spline animation curves with free-float beat duration (no quantization) — *Validated Phase 7*
- [x] **ANIM-03**: Animatable properties: size, hue, saturation, lightness — *Validated Phase 7 (rotation deferred)*
- [x] **ANIM-04**: Curves loop independently; polyrhythm emerges from mismatched durations — *Validated Phase 7*
- [x] **ANIM-05**: Bottom animation panel with stacked multi-lane view — *Validated Phase 7*
- [x] **ANIM-06**: Draggable divider between canvas and animation panel — *Validated Phase 7*

### Shape Panel

- [x] **PANL-01**: Shape edit panel exposes HSV color picker — *Validated Phase 4*
- [x] **PANL-02**: Shape edit panel exposes size slider (controls base amplitude/loudness) — *Validated Phase 4*
- [x] **PANL-03**: Shape edit panel exposes animation rate slider — *Superseded: replaced by Animate button + per-curve beat duration in Phase 7; PANL-03 original intent (configurable animation rate) fulfilled*

### Playback

- [x] **PLAY-01**: User can start and stop all audio playback — *Validated Phase 5*
- [x] **PLAY-02**: BPM/tempo is configurable via UI control — *Validated Phase 5*
- [x] **PLAY-03**: Master volume control is available — *Validated Phase 5*
- [x] **PLAY-04**: BPM range is 40–200 BPM — *Validated Phase 5*
- [x] **PLAY-05**: Key/scale selector constrains valid pitches (6 scales) — *Validated Phase 6*
- [x] **PLAY-06**: Scale mode and chromatic mode selectable per session — *Validated Phase 6 (chromatic = no quantization option)*

### Composition

- [ ] **COMP-01**: Undo/redo with minimum 50-step depth (zundo middleware) — *Deferred per D-01*
- [ ] **COMP-02**: Export canvas as PNG encoding full composition state — *Deferred per D-02*

### Non-Functional

- [ ] **PERF-01**: Canvas sustained at 60fps on mid-range hardware — *Not formally verified; subjectively smooth in testing*
- [ ] **PERF-02**: Audio latency < 20ms from trigger to audible output — *Not formally measured; Web Audio API inherently low-latency*
- [ ] **PERF-03**: Max 32 simultaneous voices without performance degradation — *Not stress-tested; 16-voice PoC limit only*
- [ ] **PERF-04**: Browser support: Chrome 120+, Firefox 120+, Safari 17+ — *Not cross-browser tested*
- [ ] **PERF-05**: First interaction to sound < 500ms after AudioContext resume — *Not formally measured*
